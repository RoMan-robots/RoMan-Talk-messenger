Умови використання RoMan Talk 2.0 UC

Створено 26.01.2024, Оновлено 04.04.2024

Ласкаво просимо до умов використання RoMan Talk! Ці умови є обов'язковими для користувачів, які бажають користуватися нашим месенжером.
Будь ласка, уважно прочитайте їх перед використанням нашого додатку.

1. Про умови

1.1. Команда RoMan Talk залишає за собою право змінювати ці умови в будь-який час без попереднього повідомлення. Зміни набувають чинності з моменту їх публікації.

1.2. Продовження використання RoMan Talk після внесення змін у ці умови означає вашу згоду з оновленою версією умов використання.

1.3. Умови можуть відрізнятися в залежності від версії месенжера.

2. Реєстрація та вхід

2.1. Для користування RoMan Talk необхідно створити обліковий запис або увійти під існуючим. Користувачі несуть відповідальність за надійність свого пароля.

2.2. Під час входу до облікового запису користувач повинен ввести свій логін та пароль, які вказав під час реєстрації.

2.3. Передавати акаунти іншим особам можна, проте власник акаунту несе відповідальність за безпеку свого облікового запису та можливі наслідки від передачі акаунту.

3. Канали

3.1. Учасники мають заборону на використання нецензурної лексики, розповсюдження матеріалів для дорослих або образливих висловлювань в каналі RoMan World Official, проте це не стосується інших каналів у разі згоди власника.

3.2. Заборонено створювати канали з образливими назвами або назвами, які містять слова для дорослих.

3.3. Спам створення каналів заборонено.

3.4. Користувачі мають можливість створювати власні канали з урахуванням вказаних правил.

3.5. Учасники каналів зобов'язані дотримуватися правил поведінки та не перешкоджати спокійному спілкуванню інших учасників.

3.6. Пам'ятайте, що в RoMan Talk користувачі різної вікової категорії, тому ви повинні відноситися до кожного з повагою.

4. Конфіденційність

4.1. RoMan Talk використовує криптографічні методи для захисту конфіденційності користувачів. Проте ми не несемо відповідальності за можливість витоку інформації через дії користувачів або технічні недоліки.

5. Відповідальність

5.1. Команда RoMan Talk залишає за собою право заблокувати обліковий запис користувача у разі порушення цих умов або негідної поведінки.

5.2. Ми можемо помилково заблокувати акаунт, проте ви можете подати на апеляцію і відновити доступ до акаунту. Підтримка розгляне вашу апеляцію та прийме рішення про розблокування або подальші дії.

5.3. У разі спаму апеляціями або писання нецензурної лексики в причині, акаунт видаляється без можливості відновлення.
